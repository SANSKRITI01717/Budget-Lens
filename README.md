# Budget-Lens
This is a corruption detector system for engineering colleges. Made for a hackathon purpose.
### Transparency in College Funding



## About
BudgetLens is a student complaint-based corruption detection 
system. Students submit Yes/No feedback on college facilities. 
If 70%+ students report a problem, the system flags potential 
misuse of allocated funds.

## What I Built
- Complete REST API backend (Node.js, Express, MongoDB)
- Student registration and login system
- Feedback collection system  
- Corruption detection algorithm (original idea and logic)
- Admin dashboard with budget risk analysis

## Tech Stack
- Backend: Node.js, Express.js, MongoDB, Mongoose
- Frontend: HTML, Bootstrap 5, Chart.js (generated using v0.dev)

## Honest Note
Backend was fully built by me from scratch.
Frontend was generated using v0.dev AI tool due to hackathon 
time constraints, then connected to the real backend API.
The corruption detection algorithm and system design is original.

## How to Run
1. Clone the repo
2. npm install
3. Create .env file with your MONGODB_URL
4. node app.js
5. Open localhost:3000
